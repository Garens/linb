Visual JS 是一个开源的 Ajax、RIA、Client/SOA 可视化快速开发工具，Visual JS 本身就是由 Visual JS 开发出来的。
特点：
1.	支持SPA，快速建立Ajax RIA Client/SOA 应用;
2.	用拖拽的方式建立复杂JavaScript界面;
3.	提供大量通用控件 (窗口、菜单和工具栏、树形表格等等);
4.	支持代码后绑定和后台装载;
5.	支持动态的从一个 js  文件中加载程序的模块;
6.	纯OOP的控件框架方便从多个维度自定义控件;
7.	支持AJAX数据源控件;
8.	支持多语言，可以在运行过程中随意切换语言;
9.	跨浏览器，支持大多数现代浏览器 (IE6+, firefox1.5+, opera9+, safari3+);
10.	生成可以直接发布的、跨平台、后台语言无关的纯 JavaScript Zip 包。

Visual JS is a Web-Based Client/SOA RAD IDE with file manager for building zero-footprint RIA that is based on a top innovative Open Source multipurpose AJAX frameworks jsLINB. Visual JS itself is developed by the Visual JS.

Features:
1.	Enables Single-Page Interfaces, allows creation of zero-footprint applications with rich user-interface;
2.	Allows development of complex GUI with AJAX using Drag 'n' Drop;
3.	Provides most common widgets (dialog, treebar, menu and toolbar, treegrid, tabs and more);
4.	Support Straight-Call and background-loading;
5.	Support dynamic loading an extra file as a component of application;
6.	Custom widgets easily with OOP UI framework;
7.	Ajax data binding supported;
8.	Support for multilingual interface: change localization strings on the fly;
9.	Cross-browser, Supports modern browsers (IE6+, firefox1.5+, opera9+, safari3+);
10.	Output pure client JavaScript code, that is platform-independent and server language-independent
