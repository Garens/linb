Thank you for choosing Sigma Visual!
Sigma visual UI builder is web based tool for AJAX RIA application UI rapid design and involved scripts programming. Written in pure JavaScript (Sigma Linb), web applications made by Sigma Visual UI builder have great compatibilities with web browsers. With this powerful builder, professional and enterprise developers can developer your web application just like what you do in VB or Delphi. And it also supports source highlight and syntax checking. Through its powerful AJAX components library (Sigma Linb), Sigma visual simplify complex and boring UI design, which artists instead of programmers are good at.
