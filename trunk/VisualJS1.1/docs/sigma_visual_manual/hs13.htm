How to install

It is quite simple to install Sigma Visual on your computer. 
System requires
l PHP5.0+
l Apache web server 1.31+

Please follow the steps to install Sigma Visual.
l Visit www.sigmawidgets.com to download zip package.
l Unzip package file to apache PHP directory, for example, %install_dir%.
l Make sure that %install_dir%/projects/ directory is writable.
l Open %install_dir%/index.html through browser.

Note that index.html is a compressed version of debug.html. If you have Sigma Visual UI builder installed
on your local machine within LAN, open debug.html to get meaningful event hander declaration and more
debug info. If you have Sigma Visual UI builder install on remote machine over internet, open index.html
to increase loading speed.
