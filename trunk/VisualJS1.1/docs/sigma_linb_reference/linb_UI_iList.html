linb.UI.iList

Instance Methods

clearItems ( key )
To clear items with specific key.
key. String. Key string.

getItems ( )
To get items array.Example - [{
"id" : "a",
"caption" : "itema",
"tips" : "item a"
},
{
"id" : "b",
"caption" : "itemb",
"tips" : "item b"
}]

returns: Array. Items array.

getListKey ( )
To get list key.
returns: String. Key string.

insertItems ( arr, base, before )
To insert items.
arr: Array. Item array.
base: Item. Existing item before or after which new items will be inserted.
before: Boolean. True for before, false for after.

removeItems ( arr )
To remove items.
arr: Array. Items array to be removed.

setItems ( v )
To set items array.Example - [{
"id" : "a",
"caption" : "itema",
"tips" : "item a"
},
{
"id" : "b",
"caption" : "itemb",
"tips" : "item b"
}]

v: Array. Items array.

setListKey ( v, flag )
To set list key.
v: String. Key string.
