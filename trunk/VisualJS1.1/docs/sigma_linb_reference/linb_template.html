linb.template

Constructor

Instance Methods

destroy ( )
To destroy itself.

empty ( )
To make inner html empty.

getDomNode ( )
To get DOM node it stands for.
return: DOM node.

render ( )
To render template to real html.
returns: The rendered html node.

renderTo ( node )
To render template to html node, and append this node to specific node as its child node.
node: Parent node.

setEvents ( key, value )
To set events by key and value. If key is object, key will be asigned to events.

setTemplate ( key, value )
To replace key with value.
