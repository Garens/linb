Profile - The first parameter passed in.

The first parameter of any event handler is profile, what can I do with profile. Profile passed in is an instance of UIProfile. Generally you can do
1) get the data model by profile.properties. For example, developer can access items by profile.properties.items.
2) get a component instance which fired the event by profile.boxing(). For example, you can change component's caption by profile.boxiing().setCaption("another caption"). That is profile.boxing() is kind of *self* in Delphi or *me* in Visual Basic.
