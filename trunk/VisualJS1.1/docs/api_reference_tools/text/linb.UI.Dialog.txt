onHotKeydown ( fun )
Fired when hot key(s) is down.
fun: function(profile, key, control, shift, alt)
profile:
key: Key which is down
control, shift, alt: boolean. Indicate whether ctrl/shift/alt key is down at the same time.

---------------------------------------
onHotKeypress ( fun )
Fired when hot key(s) is pressed.
fun: function(profile, key, control, shift, alt)
profile:
key: Key which is pressed.
control, shift, alt: boolean. Indicate whether ctrl/shift/alt key is pressed at the same time.

---------------------------------------

profile:
key: Key which is up
